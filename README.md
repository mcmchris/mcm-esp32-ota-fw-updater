# mcm-esp32-ota-fw-updater
Update your ESP32 project automatically from a GitHub repository release OTA.

When the certificates fails generate them by using:

https://openslab-osu.github.io/bearssl-certificate-utility/

And putting: `api.github.com, github.com, objects.githubusercontent.com, release-assets.githubusercontent.com, raw.githubusercontent.com,  s3.amazonaws.com` in the Domains field.
